# 프로토타이핑 3 — 문맥 브리핑 효과 검증 (피봇 후 H1)

> 가설: 브리핑을 먼저 읽으면, 같은 원문의 이해도와 완독률이 오른다.

## 설계
- 어려운 문서 1개 선정 (판결문 추천 — 원문 난독성이 높을수록 효과가 잘 보임)
- 참가자 6명을 둘로 나눔:
  - A그룹(3명): 원문만 읽음
  - B그룹(3명): 브리핑 먼저 → 원문 읽음
- 측정:
  1. 완독 여부 + 읽는 데 걸린 시간
  2. 이해도 퀴즈 3문항 (핵심 쟁점 / 결론 / 왜 그런 결론인지)
  3. "다시 이런 문서를 읽어야 한다면 이 도구를 쓰겠는가" (1~5)
- 주의: 브리핑이 퀴즈 답을 직접 알려주면 실험이 오염됨 → 결론 스포 금지 원칙이 지켜졌는지 먼저 확인

## 판정
- B그룹 이해도가 명확히 우위 → H1 통과, 대상 문서 확장(공시·논문)
- 차이 없음 → 브리핑 구성 재검토 (개념 카드가 겉도는지, 읽기 경로가 실제로 쓰이는지 관찰)
