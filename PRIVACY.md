# 갈피 크롬 확장 개인정보처리방침 (Privacy Policy)

최종 수정: 2026-07-16

**갈피 — 읽기 전 배경지식 브리핑** 확장은 개인정보를 수집하지 않습니다.

## 수집·저장하는 정보

- **없습니다.** 이름, 이메일, 브라우징 기록, 쿠키, 식별자 등 어떤 개인정보도 수집하지 않습니다.
- 확장이 로컬에 저장하는 것은 **사용자가 설정한 서버 주소 문자열 하나**뿐입니다
  (`chrome.storage.local`, 기기 밖으로 전송되지 않음).

## 기사 본문의 처리

- 사용자가 "이 기사 갈피 잡기"를 누른 경우에만, 현재 페이지의 **기사 본문 텍스트**가
  **사용자가 지정한 갈피 서버**(기본값: 사용자 본인 컴퓨터의 `http://localhost:8000`)로 전송됩니다.
- 서버는 본문을 브리핑 생성 파이프라인의 입력으로만 사용하며 **어디에도 저장하지 않습니다**
  (데이터베이스·파일 저장 없음 — 프로젝트 절대 원칙).
- "광고 없이 읽기" 리더 뷰의 본문·이미지는 `chrome.storage.session`(메모리 전용)으로만 전달되며
  브라우저를 닫으면 소멸합니다. 이미지는 원본 URL을 참조만 하고 복제하지 않습니다.
- 확장은 제3자 분석 도구, 광고 SDK, 원격 코드 로딩을 일절 사용하지 않습니다.

## 권한 사용 목적

| 권한 | 목적 |
|---|---|
| `activeTab`, `scripting`, 호스트 권한(`http/https`) | 현재 보고 있는 기사 페이지에서 본문 추출, 용어 형광펜 하이라이트 |
| `sidePanel` | 브리핑을 표시할 사이드패널 |
| `storage` | 서버 주소 설정(local), 리더 뷰 데이터 전달(session, 메모리 전용) |
