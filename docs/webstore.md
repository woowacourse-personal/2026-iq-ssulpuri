# 크롬 웹스토어 배포 가이드

배포 패키지는 `dist/galpi-extension-<버전>.zip` (재생성: 아래 "패키지 만들기").
아래 순서 중 **개발자 계정·업로드·제출은 계정 소유자만 할 수 있다.**

## 0. 전제 (v0.1.1 기준)

이 버전은 **로컬 갈피 서버가 필요**하다 (`uvicorn app.main:app` + 본인 `ANTHROPIC_API_KEY`).
스토어 설명에 이를 명시했다. 일반 사용자 대상 배포(서버 호스팅 + 요청 제한)는 별도 로드맵.

## 1. 패키지 만들기

```bash
cd extension && zip -rq ../dist/galpi-extension-$(python3 -c "import json;print(json.load(open('manifest.json'))['version'])").zip . -x ".*"
```

## 2. 개발자 계정 (1회)

1. https://chrome.google.com/webstore/devconsole 접속 (Google 계정)
2. 개발자 등록비 $5 결제 (1회)

## 3. 등록 — 아래 문구 복사해서 사용

**이름**: 갈피 — 읽기 전 배경지식 브리핑

**짧은 설명** (132자 이내):
> 뉴스가 어려운 건 당신 탓이 아닙니다. 기사가 전제하는 개념과 이전 사건을 읽기 전에 채워주고, 광고 없는 리더 뷰로 기사를 읽게 해줍니다.

**상세 설명**:
> 뉴스는 연재물입니다. 오늘 기사가 안 읽히는 건 대개 지난 편들을 못 봤기 때문입니다.
>
> 갈피는 기사 페이지에서 원클릭으로:
> ① 기사가 당신이 이미 안다고 전제하는 개념·이전 사건을 찾아
> ② 읽기 전 배경지식 브리핑으로 채워주고
> ③ 그 용어들을 기사 본문에 형광펜으로 표시합니다.
> ④ "광고 없이 읽기"를 누르면 브리핑과 기사 본문(이미지 포함)을 나란히 보는 리더 뷰가 열립니다.
>
> 요약 서비스가 아닙니다 — 결론을 대신 말하지 않고, 기사를 스스로 읽을 수 있게 만듭니다.
> AI가 지어낸 배경을 막기 위해 원문 대조 검증 단계를 거치며, 기사 밖 일반 지식은 배지로 구분 표시됩니다.
>
> ⚠️ 이 버전은 로컬 갈피 서버가 필요합니다 (Python + 본인의 Anthropic API 키).
> 설치·실행 방법: https://github.com/woowacourse-personal/2026-iq-galpi
> 기사 본문은 어디에도 저장되지 않습니다.

**카테고리**: 생산성 / **언어**: 한국어

**개인정보처리방침 URL**: 레포의 `PRIVACY.md` 공개 URL
(예: `https://github.com/woowacourse-personal/2026-iq-galpi/blob/main/PRIVACY.md` — **레포가 public이어야 함**)

**권한 사유 (심사 폼의 Justification 란)**:
- `activeTab`/`scripting`: 사용자가 요청한 순간에만 현재 기사 페이지에서 본문 텍스트를 추출하고, 브리핑의 개념 용어를 본문에 하이라이트하기 위해 필요.
- 호스트 권한(`http://*/*`, `https://*/*`): 뉴스 기사는 임의의 언론사 도메인에 있으므로, 사용자가 보고 있는 기사 페이지에서 본문 추출·하이라이트가 동작하려면 필요. 수집·전송은 사용자가 버튼을 누른 페이지의 본문 텍스트만, 사용자가 지정한 서버로만 이뤄짐.
- `sidePanel`: 브리핑 표시 UI.
- `storage`: 서버 주소 설정(local) 및 리더 뷰 데이터 전달(session, 메모리 전용).
- 원격 코드 미사용 (Readability.js는 패키지에 번들, remote hosted code 없음).

**스크린샷 (1280×800, 1~5장)** — 직접 캡처 필요:
1. 기사 페이지 + 사이드패널 브리핑(리본·개념 카드)
2. 본문 형광펜 하이라이트가 보이는 장면
3. "광고 없이 읽기" 리더 뷰 2단 화면
4. 진행 램프(①~④) 로딩 장면

## 4. 제출 후

- 넓은 호스트 권한 때문에 심사가 수 일~2주 걸릴 수 있다 (in-depth review 대상).
- 반려 시 사유가 메일로 오며, 대부분 권한 사유 보강으로 해결된다.
